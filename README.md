# Task Management System

A backend application inspired by task management platforms such as Trello.

## Goals

This project is being developed to demonstrate backend engineering skills including:

* Java 21
* Spring Boot
* PostgreSQL
* Docker
* AWS
* REST APIs
* Authentication & Authorization
